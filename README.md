## **LarLogger**

## 介绍

针对于.NET的原生日志扩展，其中包括以下功能

*   [x] 格式化支持Txt/Json
*   [x] 异步日志输出
*   [x] RGB色彩输出
*   [x] 持久化日志处理
*   [x] 以及除了原生的支持外的很多
*   [x] ...
